# Endpoint Automation Scripts

A curated collection of PowerShell scripts, Graph API utilities, and Infrastructure-as-Code templates for managing modern enterprise endpoints and cloud environments.

## Categories

| Folder | Description |
|--------|-------------|
| `intune/` | Microsoft Intune device management, compliance, and app deployment scripts |
| `autopilot/` | Windows Autopilot provisioning and hardware hash utilities |
| `azure/` | Azure resource management, Bicep templates, and Entra ID automation |
| `defender/` | Microsoft Defender for Endpoint configuration and monitoring |
| `monitoring/` | Endpoint health checks, NinjaOne integrations, and KQL queries |
| `logs/` | Daily automation run logs |

## Tech Stack
- PowerShell 7+
- Microsoft Graph API
- Azure Bicep
- KQL (Kusto Query Language)
